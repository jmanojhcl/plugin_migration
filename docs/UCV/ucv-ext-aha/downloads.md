# Aha! - Downloads

## Docker pull command

```docker pull hclcr.io/accelerate/ucv-ext-aha:1.0.1```

## Docker image tar files
- [ucv-ext-aha:1.0.1.tar.7z.001](https://hclcr.io/files/devops/accelerate/plugins/ucv-ext-aha/ucv-ext-aha%3A1.0.1.tar)


## Instructions to load plugin image from tar file:

- Load the docker image from tar file using: ```docker load <tar_file_name>```
- Go to **Velocity** -> **Settings** -> **Integrations** -> **Available** -> **Load plugin** -> Added the image name and click **Submit**
