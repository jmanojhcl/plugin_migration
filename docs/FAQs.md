# Frequently Asked Questions

Connect your HCL Software solutions with the rest of your tech stack. Find your plugins for issue tracking, continuous delivery, automated testing, security scanning and more.

**1. Are these plugins free?**

   The download and installation of these plugins is totally free. Some tools might require a paid license, but the plugins themselves do not require payment.

**2. How to install the plugins?**

   Please refer: Installing HCL Velocity plugins, Installing HCL Deploy plugins.

**3. What can the plugins on this site do?**

   Plugins create a seamless connection between your HCL Software solutions and other software in your toolchain. The plugins in this library cover a wide range of tools and functions – from automated testing to source control – that help you do more with your HCL Software solutions.

**4. Are these plugins secure?**

   Absolutely. HCL Software prioritizes security across our entire product suite. Learn more in the [HCL Software Trust Center](https://www.hcltechsw.com/resources/trust-center).

**5. Is there someone I can reach out to for plugins questions and support?**

   You can reach us by filling out the form on this [page](https://www.hcltechsw.com/wps/portal/contact-us). You can also explore our [Knowledge Center](https://support.hcltechsw.com/csm) for product documentation and community forums.

**6. I don’t see the plugin I want here. Can I request a custom plugin?**

   Yes! We welcome feedback from our users on what plugins we should develop next and will work with you to develop custom plugins. Connect with us by emailing [DevOpsInfo@hcl.com](mailto:DevOpsInfo@hcl.com).

**7. How do I know when a plugin has been updated?**

   Sign up for our monthly plugins newsletter to keep up with the latest plugin updates and new plugins that have been added to our library. Sign up here.
